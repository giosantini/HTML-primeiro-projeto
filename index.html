Historia dos computadores

Com o passar dos anos os computadores foram evoluindo, antes era usado apenas para calculos,
com somente o exército tendo acesso a essa tecnologia. E a evolução seguiu chegando a terceira geração (1965 a 1970),
quando surgiu os primeiros computadores comercializados, com circuitos integrados, capacitores, transistores e resistores. De 1971 a 2000, veio o primeiro mouse e perifericos.
E de 2000 aos dias atuais temos inteligência artificial, criptomoedas, chatGPT etc. Fora muitas coisas que ainda podem surgir sendo em uma realidade não muito distante.

Surgimento da internet

Na década de 1950, na guerra fria foi criada a DARPA (Defense Advanced Research Projects Agency).
Fundada pelo governo americano com intuito de realizar pesquisas para fins militares, na mesma década o cientista da computação e psicologo,
Joseph Carl Robnett Licklider teorizou sobre uma rede de computadores onde todos poderiam se comunicar através dela.
E no ano de 1969, foi construida a primeira rede de computadores para fins de transmição de dados militares e sigilosos,
e fazer interligação entre os departamentos de pesquisa dos EUA. Seguindo no ano de 1989 foi criado www (World Wide Web)
com intuito de comercializar a internet com as pessoas, surgindo o primeiro navegador de internet o Netscape.

O que é client

O termo client, é usado para definir o aparelho que recebe o conteudo do servidor para o nevagador, que entende os códigos da linguagem de programação que são HTML, CSS e javascript.

O que é server

O server, funciona como uma ponte entre o client o dispositivo, e o usuario onde trocam informações, arquivos etc,
como tambem através do navegador faz uma requisição HTTP para o server, e por sua vez o server retorna arquivos ou HTML, CSS e javascript.

Linguagens de programação

São conjuntos de regras e instruções, com liguagem escrita e formal para gerar um programa (software).
Existe as liguagens server-side, que são liguagens que são executadas do lado do servidor
como: PHP, ruby, python, javascript, etc. O javascript é uma liguagem tanto server-side quanto client-side,
por ter o node.js que entende o motor V8 presente nos navegadores.
